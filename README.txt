# BehaviourMSL
Behaviour MSL contiene:
- Data sin analizar de experimentos MSL de Flor (FJ), Guille (GG) y Guada (GC)
- Las funciones utilizadas en el paper de flor para analizar la data Id_runAnalysis_FJ y Id_runAnalysisGroup_FJ para el análisis grupal (El FJ indica que utiliza parámetros de Id_runAnalysis_FJ, pero es un script de GC). 
- Para utilizar dichas funciones, se las llama desde LoadSubjects, donde se cargan todos los resultados de los sujetos ya analizados de forma individual.
